## General info
This project aims to hold french politicians to a higher standard by making them accountable, comparing their tweets with their actual votes in parliament. 

## Technologies
Project created with: 
- Ruby on Rails 6
- JS ES6
- Bootstrap v.5
- HTML&SCSS
